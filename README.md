# Vite-React-TS-Starter

This an somewhat opinionated template to quickly build new projects. List of stuff included:
- React 18
- Vite
- Typescript
- ESLint
- Prettier
- Tailwind CSS

## Usage

`npm clone https://github.com/artem-tarasenko/Vite-React-TS-Starter.git *foldername* `

`cd *foldername*`

`npm i`

`npm start`

See localhost:3000
